# docs

현재 코드 기준으로 유지하는 운영 문서 목록이다.

## 유지 문서

- `capture-process-description.md`: `receiver capture` 파이프라인 설명
- `deploy-receiver.md`: `receiver` 배포 산출물과 권장 실행 방식
- `deploy-sender.md`: `sender` 배포 산출물과 권장 실행 방식
- `warning.ko.md`: 사용 전 반드시 확인해야 하는 강한 경고 문서
- `warning.en.md`: `warning` 의 영문 버전
- `slide.md`: `sender slide` 실행 방식과 단축키 안내
- `faster.md`: `slide` / `capture` 성능 튜닝 메모
- `todo.md`: 현재 남아 있는 작업만 정리한 TODO

## 정리 원칙

- 문서는 현재 `apps/*`, `libs/*` 구조와 실제 CLI 기준으로 유지한다.
- 이미 끝난 작업 기록이나 구현 전 설계 초안은 운영 문서에 남기지 않는다.
- 과거 구조 검토 문서는 별도 보관 가치가 없으면 삭제한다.
